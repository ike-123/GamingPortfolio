Bedroom Game

Engine: Unity

Platform: PC

Project Start Date: 4th November 2021

Project End Date: 30th November 2021

Link

This was a fun little project I worked on, The Idea is you are an action figure In a kids bedroom, surrounded by toys. Players have complete freedom to interact with their environment, whether it’s blasting toys across the room or stacking them to build something unique and imaginative.

One achievement I’m most proud of in this game was creating the block picker tool. This uses a Raycast to detect when a block is selected, followed by a SphereCast to check if it's in close proximity to another block. If a nearby block is detected, the selected block snaps into place next to it. This makes it really easy and intuitive for players to start building.

Skills learnt: Inverse kinematics, Rigidbody movement, Physics OverlapSphere,

What's Real?

Engine: Unity

Platform: PC

Project Start Date: February 20th 2022

Project End Date: February 27th 2022

Link

What’s Real was my first ever Game Jam game. This was part of the 2022 Brackeys Game Jam and the theme was “It is not Real”. The Idea I had was to make a fun platformer, with a slight twist being some of the platforms were holograms(fake) . Players had to identify and avoid stepping on the fake platforms to be able to progress to the next level. The game jam lasted for a week and I decided to do this solo.

Towards the end of the jam, I ran into several challenges, primarily due to overscoping the project. A lot of the features I wanted to add proved unfeasible given the limited amount of time that I had. This forced me to, remove and rework a lot of my gameplay to make sure everything was still coherent. As a result, I was making last-minute adjustments right up until the deadline.

In the final moments of the jam, the pressure led to a mistake, as I accidentally submitted the game to the wrong submission link, just seconds before the deadline. By the time I realised the mistake, it was too late to fix it.

Although the Game Jam didn’t go as I wanted it to. It taught me a valuable lesson about choosing the correct scope for your project and having effective time management.

Skills Learnt:Rapid prototyping and iteration, Creating fun gameplay with a simple mechanic

Death Quest

Engine: Unity

Platform: PC

Project Start Date: 20th May 2022

Project End Date: 30th May 2022

Link

Disappointed in the result of my last Jam. I joined another Game Jam, with the intention of learning from all of the mistakes I made during the first one. This Game Jam was hosted by GameDev.TV and the theme was “Death is only the beginning”. After spending a couple hours thinking of a game mechanic I decided it was going to be an escape room platformer which made dying an integral part of the gameplay. Having done some planning I knew this would be feasible to create within the duration of the jam which was 10 days.

Overall the Game Jam was quite successful. Out of the 922 entries my game was ranked 26th. I was thrilled with the result and was glad to see my determination pay off.

Skills Learnt: Developing a game under strict time constraints, Rapid prototyping and iteration, Creating fun gameplay with a simple mechanic, C# Interfaces
